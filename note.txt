===== Buổi 3 =====
* communication
    - Props: truyền dữ liệu từ component cha sang component con
        + class: Nhận props thông qua con trỏ this => this.props
        + function: Nhận props thông qua tham số của function(props)
                    Không còn tồn tại từ khoá this
        + children: Nhận tất cả từ Component cha. (props.children)
    - Lifting State Up: truyền dữ liệu từ component con sang component cha
            Component cha truyền vào Component con 1 props 
            => props mang hàng (hàm này giúp component cha nhận được data từ component con)
